# 14.2 useRenderCount

## 1. 实现思路

统计组件渲染次数。

## 2. 代码实现

```typescript
import { useRef, useEffect } from 'react';

function useRenderCount() {
  const count = useRef(1);

  useEffect(() => {
    count.current++;
  });

  return count.current;
}

export default useRenderCount;
```

## 3. 使用示例

```tsx
const renderCount = useRenderCount();
console.log('Render count:', renderCount);
```
