# 14.3 usePrevious

## 1. 实现思路

获取上一次渲染时的值。

## 2. 代码实现

```typescript
import { useEffect, useRef } from 'react';

function usePrevious<T>(value: T): T | undefined {
  const ref = useRef<T>();

  useEffect(() => {
    ref.current = value;
  }, [value]);

  return ref.current;
}

export default usePrevious;
```

## 3. 使用示例

```tsx
const prevCount = usePrevious(count);
```

## 4. 常见问题

- **执行时机**：useEffect 在渲染后执行，因此 ref.current 保存的是上一次的值。
