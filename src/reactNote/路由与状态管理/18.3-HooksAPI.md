# 18.3 Hooks API


React Router v6 全面拥抱 Hooks。

## 1. useNavigate

编程式导航。

```tsx
import { useNavigate } from 'react-router-dom';

function LoginButton() {
  const navigate = useNavigate();
  return <button onClick={() => navigate('/home')}>Login</button>;
}
```

**实现思路**：直接从 Context 中获取 `history.push` / `history.replace` 方法。

## 2. useParams

获取路径参数（如 `/user/:id` 中的 `id`）。

```tsx
// Route path="/user/:id"
import { useParams } from 'react-router-dom';

function User() {
  const { id } = useParams();
  return <div>User ID: {id}</div>;
}
```

**实现思路**：使用 `matchPath` 匹配当前 `location.pathname` 和路由规则，提取参数。

## 3. useLocation

获取当前路由对象。

```tsx
const location = useLocation();
console.log(location.pathname);
console.log(location.search);
console.log(location.state);
```

## 4. useSearchParams

处理查询字符串（Query String）。

```tsx
const [searchParams, setSearchParams] = useSearchParams();
const q = searchParams.get('q');

// 更新查询参数
setSearchParams({ q: 'react' });
```

## 5. useMatch

检测当前路径是否匹配某个模式。

```tsx
const match = useMatch('/user/:id');
if (match) {
  // ...
}
```
