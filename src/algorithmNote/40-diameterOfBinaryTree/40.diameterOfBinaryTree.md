# 二叉树的直径 (Diameter of Binary Tree)

## 1. 题目描述

给定一棵二叉树，你需要计算它的直径长度。一棵二叉树的直径长度是任意两个结点路径长度中的最大值。这条路径可能穿过也可能不穿过根结点。

**注意：** 两结点之间的路径长度是以它们之间边的数目表示。

**示例 :**
给定二叉树

```
          1
         / \
        2   3
       / \     
      4   5    
```

返回 **3**, 它的长度是路径 [4,2,1,3] 或者 [5,2,1,3]。

## 2. 实现思路

二叉树的直径不一定经过根节点。
对于每一个节点，经过该节点的最长路径长度 = **左子树的最大深度** + **右子树的最大深度**。

我们的目标是找到全局最大的这个值。

我们可以使用 **后序遍历**（DFS）来解决这个问题：
1.  定义一个全局变量 `ans` 来记录最大直径。
2.  定义一个递归函数 `depth(node)`，计算以 `node` 为根的子树的深度（最大节点数）：
    *   如果 `node` 为空，深度为 0。
    *   递归计算左子树深度 `L = depth(node.left)`。
    *   递归计算右子树深度 `R = depth(node.right)`。
    *   更新全局最大直径：`ans = Math.max(ans, L + R)`。
    *   返回当前节点的最大深度：`Math.max(L, R) + 1`。
3.  最终返回 `ans`。

## 3. 代码实现 (TypeScript)

```typescript
/**
 * Definition for a binary tree node.
 * class TreeNode {
 *     val: number
 *     left: TreeNode | null
 *     right: TreeNode | null
 *     constructor(val?: number, left?: TreeNode | null, right?: TreeNode | null) {
 *         this.val = (val===undefined ? 0 : val)
 *         this.left = (left===undefined ? null : left)
 *         this.right = (right===undefined ? null : right)
 *     }
 * }
 */

function diameterOfBinaryTree(root: TreeNode | null): number {
    let ans = 0;

    function depth(node: TreeNode | null): number {
        if (node === null) {
            return 0; // 空节点深度为 0
        }
        
        const L = depth(node.left);  // 左子树深度
        const R = depth(node.right); // 右子树深度
        
        // 更新最大直径：穿过当前节点的路径长度 = L + R
        ans = Math.max(ans, L + R);
        
        // 返回以当前节点为根的子树深度
        return Math.max(L, R) + 1;
    }

    depth(root);
    return ans;
}
```

## 4. 复杂度分析

*   **时间复杂度**: $O(n)$。其中 $n$ 是二叉树的节点数。我们需要遍历每个节点一次。
*   **空间复杂度**: $O(height)$。递归栈的深度取决于树的高度。
