# 73. 字符串解码

## 题目描述

给定一个经过编码的字符串，返回它解码后的字符串。

编码规则为: `k[encoded_string]`，表示其中方括号内部的 `encoded_string` 正好重复 `k` 次。注意 `k` 保证为正整数。

你可以认为输入字符串总是有效的；输入字符串中没有额外的空格，且输入的方括号总是符合格式要求的。

此外，你可以认为原始数据不包含数字，所有的数字只表示重复的次数 `k` ，例如不会出现像 `3a` 或 `2[4]` 的输入。

**示例 1：**

```text
输入：s = "3[a]2[bc]"
输出："aaabcbc"
```

**示例 2：**

```text
输入：s = "3[a2[c]]"
输出："accaccacc"
```

**示例 3：**

```text
输入：s = "2[abc]3[cd]ef"
输出："abcabccdcdcdef"
```

**示例 4：**

```text
输入：s = "abc3[cd]xyz"
输出："abccdcdcdxyz"
```

## 实现思路

这是一个经典的辅助栈应用问题。因为括号内可能包含括号（嵌套结构），所以非常适合用栈来解决。

我们需要两个栈：
1. `stackMulti`：存储重复次数 `multi`。
2. `stackRes`：存储当前括号前的字符串 `res`。

遍历字符串 `s`：
1. 如果遇到 `[`：
   - 将当前的 `multi` 和 `res` 分别压入栈中。
   - 重置 `multi` 为 0，`res` 为空字符串。
2. 如果遇到 `]`：
   - 弹出 `stackMulti` 的栈顶元素，记为 `curMulti`。
   - 弹出 `stackRes` 的栈顶元素，记为 `lastRes`。
   - 将当前 `res` 重复 `curMulti` 次，并拼接到 `lastRes` 后面。
   - 更新 `res` 为拼接后的结果。
3. 如果遇到数字：
   - 更新 `multi`：`multi = multi * 10 + parseInt(c)`（处理多位数字）。
4. 如果遇到普通字符：
   - 将字符拼接到 `res` 后面。

遍历结束后，`res` 即为最终结果。

## 代码实现

```typescript
export function decodeString(s: string): string {
  const stackMulti: number[] = [];
  const stackRes: string[] = [];
  let multi = 0;
  let res = '';

  for (const c of s) {
    if (c === '[') {
      stackMulti.push(multi);
      stackRes.push(res);
      multi = 0;
      res = '';
    } else if (c === ']') {
      let tmp = '';
      const curMulti = stackMulti.pop();
      if (curMulti !== undefined) {
        for (let i = 0; i < curMulti; i++) {
          tmp += res;
        }
      }
      res = (stackRes.pop() || '') + tmp;
    } else if (c >= '0' && c <= '9') {
      multi = multi * 10 + parseInt(c);
    } else {
      res += c;
    }
  }
  
  return res;
}
```

## 复杂度分析

- **时间复杂度**：O(n)，其中 n 是解码后字符串的长度。我们需要遍历字符串并构造结果。
- **空间复杂度**：O(n)，最坏情况下（例如嵌套层数很深），栈的深度可能达到 O(n)。
