# 将有序数组转换为二叉搜索树 (Convert Sorted Array to Binary Search Tree)

## 1. 题目描述

给你一个整数数组 `nums` ，其中元素已经按 **升序** 排列，请你将其转换为一棵 **高度平衡** 二叉搜索树。

**高度平衡** 二叉树是一棵满足「每个节点的左右两个子树的高度差的绝对值不超过 1 」的二叉树。

**示例 1：**

![bst](https://assets.leetcode.com/uploads/2021/02/18/btree1.jpg)

> **输入：** nums = [-10,-3,0,5,9]  
> **输出：** [0,-3,9,-10,null,5]  
> **解释：** [0,-10,5,null,-3,null,9] 也将被视为正确答案：  
> ![bst2](https://assets.leetcode.com/uploads/2021/02/18/btree2.jpg)

**示例 2：**

> **输入：** nums = [1,3]  
> **输出：** [3,1]  
> **解释：** [1,null,3] 和 [3,1] 都是高度平衡二叉搜索树。

## 2. 实现思路

二叉搜索树（BST）的中序遍历是升序的。给定一个升序数组，要构造一棵高度平衡的 BST，我们可以选择 **数组的中间元素** 作为根节点。
这样，数组左边的元素构成左子树，右边的元素构成右子树。由于左右两边的元素数量差不超过 1，因此构造出的树一定是高度平衡的。

这是一个典型的 **分治 (Divide and Conquer)** 问题：
1.  取数组中间的元素 `mid` 作为当前子树的根节点。
2.  递归地用 `mid` 左边的子数组构造左子树。
3.  递归地用 `mid` 右边的子数组构造右子树。

## 3. 代码实现 (TypeScript)

```typescript
/**
 * Definition for a binary tree node.
 * class TreeNode {
 *     val: number
 *     left: TreeNode | null
 *     right: TreeNode | null
 *     constructor(val?: number, left?: TreeNode | null, right?: TreeNode | null) {
 *         this.val = (val===undefined ? 0 : val)
 *         this.left = (left===undefined ? null : left)
 *         this.right = (right===undefined ? null : right)
 *     }
 * }
 */

function sortedArrayToBST(nums: number[]): TreeNode | null {
    return buildBST(nums, 0, nums.length - 1);
}

function buildBST(nums: number[], left: number, right: number): TreeNode | null {
    // 递归终止条件：区间为空
    if (left > right) {
        return null;
    }

    // 选择中间位置作为根节点
    // 这里的 mid 计算方式会选择中间或者中间偏左的元素
    const mid = Math.floor((left + right) / 2);
    const root = new TreeNode(nums[mid]);

    // 递归构建左右子树
    root.left = buildBST(nums, left, mid - 1);
    root.right = buildBST(nums, mid + 1, right);

    return root;
}
```

## 4. 复杂度分析

*   **时间复杂度**: $O(n)$。其中 $n$ 是数组的长度。每个数字只会被访问一次。
*   **空间复杂度**: $O(\log n)$。递归栈的深度取决于树的高度，对于平衡二叉树，高度为 $\log n$。
