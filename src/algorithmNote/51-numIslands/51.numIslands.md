# 200. 岛屿数量

## 题目描述

给你一个由 `'1'`（陆地）和 `'0'`（水）组成的的二维网格，请你计算网格中岛屿的数量。

岛屿总是被水包围，并且每座岛屿只能由水平方向和/或竖直方向上相邻的陆地连接形成。

此外，你可以假设该网格的四条边均被水包围。

## 实现思路

我们可以遍历整个网格。如果当前格子是 `'1'`（陆地），说明我们发现了一个新岛屿，岛屿数量加 1。
然后，我们需要把这个岛屿“铲平”，即把当前格子以及所有与它相连（上下左右）的陆地都标记为 `'0'`（水），防止重复计算。

这个过程可以使用 **深度优先搜索 (DFS)** 或 **广度优先搜索 (BFS)** 来实现。这里使用 DFS，代码更简洁。

**DFS 流程**：
1. 遍历网格的每个格子 `(i, j)`。
2. 如果 `grid[i][j] == '1'`：
   - 计数器 `count` 加 1。
   - 调用 `dfs(grid, i, j)` 进行感染/标记。
3. `dfs` 函数：
   - 边界检查：如果越界或当前格子是 `'0'`，直接返回。
   - 标记：将当前格子设为 `'0'`。
   - 递归：对上下左右四个邻居调用 `dfs`。

## 代码实现

```typescript
export function numIslands(grid: string[][]): number {
  if (!grid || grid.length === 0) {
    return 0;
  }

  const m = grid.length;
  const n = grid[0].length;
  let count = 0;

  for (let i = 0; i < m; i++) {
    for (let j = 0; j < n; j++) {
      if (grid[i][j] === '1') {
        count++;
        // 启动 DFS，将当前岛屿的所有陆地置为 '0'
        dfs(grid, i, j, m, n);
      }
    }
  }

  return count;
}

function dfs(grid: string[][], i: number, j: number, m: number, n: number) {
  // 边界检查 或 遇到水
  if (i < 0 || j < 0 || i >= m || j >= n || grid[i][j] === '0') {
    return;
  }

  // 沉岛：将当前陆地标记为水，避免重复访问
  grid[i][j] = '0';

  // 访问上下左右四个方向
  dfs(grid, i - 1, j, m, n); // 上
  dfs(grid, i + 1, j, m, n); // 下
  dfs(grid, i, j - 1, m, n); // 左
  dfs(grid, i, j + 1, m, n); // 右
}
```

## 复杂度分析

- **时间复杂度**：O(M * N)，其中 M 和 N 分别为行数和列数。每个格子最多被访问两次（一次在主循环中，一次在 DFS 中）。
- **空间复杂度**：O(M * N)，最坏情况下（全为陆地），DFS 调用栈的深度可能达到 M * N。
