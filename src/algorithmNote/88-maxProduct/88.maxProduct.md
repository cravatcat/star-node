# 88. 乘积最大子数组

## 题目描述

给你一个整数数组 `nums` ，请你找出数组中乘积最大的非空连续子数组（该子数组中至少包含一个数字），并返回该子数组所对应的乘积。

测试用例的答案是一个 **32-位** 整数。

**示例 1:**

```text
输入: nums = [2,3,-2,4]
输出: 6
解释: 子数组 [2,3] 有最大乘积 6。
```

**示例 2:**

```text
输入: nums = [-2,0,-1]
输出: 0
解释: 结果不能为 2, 因为 [-2,-1] 不是子数组。
```

## 实现思路

这道题和“最大子数组和”非常类似，但是乘法有一个特点：**负负得正**。
也就是说，当前位置的最小值，乘上一个负数，可能会变成最大值。

因此，我们在遍历时，需要同时维护两个变量：
1. `maxF`：以当前元素结尾的乘积最大子数组的乘积。
2. `minF`：以当前元素结尾的乘积最小子数组的乘积。

状态转移方程：
- `maxF[i] = max(nums[i], maxF[i-1] * nums[i], minF[i-1] * nums[i])`
- `minF[i] = min(nums[i], minF[i-1] * nums[i], maxF[i-1] * nums[i])`

我们只需要常数个变量来维护 `maxF` 和 `minF`。

## 代码实现

```typescript
export function maxProduct(nums: number[]): number {
  let maxF = nums[0];
  let minF = nums[0];
  let ans = nums[0];

  for (let i = 1; i < nums.length; i++) {
    const mx = maxF, mn = minF;
    maxF = Math.max(mx * nums[i], Math.max(nums[i], mn * nums[i]));
    minF = Math.min(mn * nums[i], Math.min(nums[i], mx * nums[i]));
    ans = Math.max(maxF, ans);
  }

  return ans;
}
```

## 复杂度分析

- **时间复杂度**：O(n)，只需要遍历一次数组。
- **空间复杂度**：O(1)，只需要常数个变量。
