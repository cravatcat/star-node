# 82. 最长回文子串

## 题目描述

给你一个字符串 `s`，找到 `s` 中最长的回文子串。

**示例 1：**

```text
输入：s = "babad"
输出："bab"
解释："aba" 同样是符合题意的答案。
```

**示例 2：**

```text
输入：s = "cbbd"
输出："bb"
```

## 实现思路

**中心扩散法**：

回文串有一个中心，我们可以枚举这个中心，然后向两边扩展。
需要注意的是，回文串的长度可能是奇数（中心是一个字符），也可能是偶数（中心是两个字符）。
所以我们需要枚举 `2n - 1` 个中心（`n` 个字符和 `n-1` 个空隙）。

对于每个中心，我们向两边扩展，直到不满足回文条件或者到达边界。记录最长的回文子串即可。

## 代码实现

```typescript
export function longestPalindrome(s: string): string {
  if (s.length < 2) {
    return s;
  }

  let start = 0;
  let maxLength = 1;

  function expandAroundCenter(left: number, right: number) {
    while (left >= 0 && right < s.length && s[left] === s[right]) {
      if (right - left + 1 > maxLength) {
        start = left;
        maxLength = right - left + 1;
      }
      left--;
      right++;
    }
  }

  for (let i = 0; i < s.length; i++) {
    expandAroundCenter(i, i); // 奇数长度
    expandAroundCenter(i, i + 1); // 偶数长度
  }

  return s.substring(start, start + maxLength);
}
```

## 复杂度分析

- **时间复杂度**：O(n^2)，枚举中心需要 O(n)，扩展需要 O(n)。
- **空间复杂度**：O(1)，只需要常数个变量。
