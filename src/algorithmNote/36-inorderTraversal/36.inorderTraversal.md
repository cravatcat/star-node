# 二叉树的中序遍历 (Binary Tree Inorder Traversal)

## 1. 题目描述

给定一个二叉树的根节点 `root` ，返回它的 **中序** 遍历。

**示例 1:**

> **输入:** root = [1,null,2,3]  
> **输出:** [1,3,2]

**示例 2:**

> **输入:** root = []  
> **输出:** []

**示例 3:**

> **输入:** root = [1]  
> **输出:** [1]

## 2. 实现思路

中序遍历的顺序是：**左子树 -> 根节点 -> 右子树**。

通常有两种实现方式：

### 方法一：递归 (Recursion)
这是最直观的方法。定义一个递归函数，先递归调用左子树，然后访问根节点，最后递归调用右子树。

### 方法二：迭代 (Iteration)
我们可以使用 **栈 (Stack)** 来模拟递归的过程。
递归的本质就是系统调用栈，我们手动维护一个栈，可以达到同样的效果，并且在某些场景下（如防止栈溢出）更可控。

## 3. 详细步骤 (迭代法)

1.  初始化一个空数组 `res` 用于存储结果，一个空栈 `stack` 用于辅助遍历。
2.  初始化指针 `curr` 指向根节点 `root`。
3.  当 `curr` 不为空，或者 `stack` 不为空时，循环执行：
    *   **不断向左深入**：如果 `curr` 不为空，将 `curr` 入栈，并让 `curr = curr.left`，直到没有左子节点。
    *   **弹出并访问**：当 `curr` 为空（说明左边走到头了），从 `stack` 中弹出一个节点赋给 `curr`。将 `curr.val` 加入结果数组 `res`。
    *   **转向右子树**：让 `curr = curr.right`，继续下一轮循环。
4.  返回结果数组 `res`。

## 4. 代码实现 (TypeScript)

```typescript
/**
 * Definition for a binary tree node.
 * class TreeNode {
 *     val: number
 *     left: TreeNode | null
 *     right: TreeNode | null
 *     constructor(val?: number, left?: TreeNode | null, right?: TreeNode | null) {
 *         this.val = (val===undefined ? 0 : val)
 *         this.left = (left===undefined ? null : left)
 *         this.right = (right===undefined ? null : right)
 *     }
 * }
 */

function inorderTraversal(root: TreeNode | null): number[] {
    const res: number[] = [];
    const stack: TreeNode[] = [];
    let curr = root;

    while (curr !== null || stack.length > 0) {
        // 1. 尽可能向左走，把沿途的节点入栈
        while (curr !== null) {
            stack.push(curr);
            curr = curr.left;
        }
        
        // 2. 左边走到底了，弹出栈顶节点（即当前的“根”节点）
        curr = stack.pop()!;
        res.push(curr.val);
        
        // 3. 转向右子树
        curr = curr.right;
    }

    return res;
}
```

## 5. 复杂度分析

*   **时间复杂度**: $O(n)$。其中 $n$ 是二叉树的节点数。每一个节点恰好被遍历一次。
*   **空间复杂度**: $O(n)$。为迭代过程中显式栈的开销，平均情况下为 $O(\log n)$，最坏情况下（树退化为链表）为 $O(n)$。
