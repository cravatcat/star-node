# 99. 汉明距离

## 题目描述

两个整数之间的 [汉明距离](https://baike.baidu.com/item/%E6%B1%89%E6%98%8E%E8%B7%9D%E7%A6%BB) 指的是这两个数字对应二进制位不同的位置的数目。

给你两个整数 `x` 和 `y`，计算并返回它们之间的汉明距离。

## 实现思路

汉明距离本质上就是两个数字异或结果中 `1` 的个数。

算法流程：
1. 计算 `xor = x ^ y`。此时 `xor` 中为 `1` 的位表示 `x` 和 `y` 在该位上不同。
2. 统计 `xor` 中 `1` 的个数。可以通过不断右移并检查最低位是否为 `1` 来实现，或者使用 Brian Kernighan 算法（`n & (n - 1)`）来消除最低位的 `1`。

这里使用最直观的右移法。

## 代码实现

```typescript
export function hammingDistance(x: number, y: number): number {
  let xor = x ^ y;
  let distance = 0;
  while (xor !== 0) {
    if (xor & 1) {
      distance++;
    }
    xor >>= 1;
  }
  return distance;
}
```

## 复杂度分析

- **时间复杂度**：O(k)，其中 k 是整数的位数（如 32 位整数则为 32）。
- **空间复杂度**：O(1)。
