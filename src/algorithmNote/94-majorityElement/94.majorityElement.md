# 94. 多数元素

## 题目描述

给定一个大小为 `n` 的数组 `nums` ，返回其中的多数元素。多数元素是指在数组中出现次数 **大于** `⌊ n/2 ⌋` 的元素。

你可以假设数组是非空的，并且给定的数组总是存在多数元素。

## 实现思路

这道题可以使用 **Boyer-Moore 投票算法**。

算法核心思想：
- 维护一个候选众数 `candidate` 和它出现的次数 `count`。
- 初始时 `count` 为 0。
- 遍历数组：
  - 如果 `count` 为 0，我们假设当前元素 `num` 是众数，将 `candidate` 更新为 `num`，并将 `count` 重置为 1。
  - 如果 `count` 不为 0，我们判断当前元素 `num` 是否等于 `candidate`：
    - 如果相等，`count` 加 1。
    - 如果不相等，`count` 减 1。
- 因为多数元素出现的次数大于 `n/2`，所以最终剩下的 `candidate` 一定是多数元素。

## 代码实现

```typescript
export function majorityElement(nums: number[]): number {
  let count = 0;
  let candidate: number | null = null;

  for (const num of nums) {
    if (count === 0) {
      candidate = num;
    }
    count += (num === candidate) ? 1 : -1;
  }

  return candidate as number;
}
```

## 复杂度分析

- **时间复杂度**：O(n)，其中 n 是数组的长度。
- **空间复杂度**：O(1)，只需要常数空间。
