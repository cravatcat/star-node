# 89. 最长有效括号

## 题目描述

给你一个只包含 `'('` 和 `')'` 的字符串，找出最长有效（格式正确且连续）括号子串的长度。

**示例 1：**

```text
输入：s = "(()"
输出：2
解释：最长有效括号子串是 "()"
```

**示例 2：**

```text
输入：s = ")()())"
输出：4
解释：最长有效括号子串是 "()()"
```

## 实现思路

我们可以使用 **栈** 来解决这个问题。

栈底始终保持一个 **“最后一个没有被匹配的右括号的下标”**。
初始时，我们将 -1 入栈。

遍历字符串：
1. 遇到 `'('`：将下标入栈。
2. 遇到 `')'`：
   - 先弹出栈顶元素（匹配最近的一个左括号）。
   - 如果栈为空，说明当前的右括号没有匹配的左括号，将当前下标入栈（更新“最后一个没有被匹配的右括号的下标”）。
   - 如果栈不为空，说明匹配成功，当前有效括号的长度为 `i - stack[top]`，更新最大长度。

## 代码实现

```typescript
export function longestValidParentheses(s: string): number {
  let maxLen = 0;
  const stack: number[] = [];
  stack.push(-1); // 放入一个参照物，表示最后一个未被匹配的右括号的下标（初始为 -1）

  for (let i = 0; i < s.length; i++) {
    if (s[i] === '(') {
      stack.push(i);
    } else {
      stack.pop();
      if (stack.length === 0) {
        stack.push(i); // 更新参照物
      } else {
        maxLen = Math.max(maxLen, i - stack[stack.length - 1]);
      }
    }
  }

  return maxLen;
}
```

## 复杂度分析

- **时间复杂度**：O(n)，遍历一次字符串。
- **空间复杂度**：O(n)，栈的大小最坏情况下为 n。
